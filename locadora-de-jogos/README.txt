Criado por Pedro Napar de Oliveira e Vinicius

Este projeto foi criado em um período de 3 semanas e
contém a funcionalidade de se conectar à um banco de dados local
e fazer adições, edições e remoções de clientes, jogos, e histórico de aluguéis

tais funcionalidades são consideradas administrativas, para um funcionário fazer login ele
pode usar o identificador unico, tal que é providênciado pelo funcionário que fez o cadastro
deste cliente.

como cliente você podera apenas alugar, editar suas próprias informações e efetuar pagamentos,
o cliente está ciente de que caso atrase para devolver o jogo ele receberá uma multa baseada no valor
total do jogo para cada dia de atraso.
